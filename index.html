Capabilities

Maximum Utility,
Minimum Space

In just ~315 lines: 4 API calls, 12+ language parsing, 8 core checks, 9 budget tiers, 10 domain scores, and ASCII report generation.

01

Multi-Language Audit Engine

Scans repositories in Go, Python, JavaScript, TypeScript, Java, C, C++, C#, Ruby, PHP, Rust, and Bash.

  • Up to 80 source files per scan
  • Auto-skips vendor / node_modules / build dirs
  • 120KB file size safety limit
  • Cross-platform regex pattern matching
02

4D Constraint Detection

Validates all Code Olympics dimensions with weighted scoring and intelligent fallbacks.

  • D1: 8 core rules (imports, functions, loops, variables)
  • D2: 9 line budget tiers (50 → 650)
  • D3: 10 domain keyword groups (×4 / ×2 / ×1 weights)
  • D4: GitHub API + file extension fallback
03

Real-Time Progress Feedback

Visual progress spinners and animated status updates during each of the 4 GitHub API phases.

  • 4-frame spinner animation (~160ms per step)
  • Phase-by-phase completion indicators
  • Graceful error handling with descriptions
  • Dual output: console + file report
04

Auto-Generated Audit Reports

Every scan produces a timestamped co-check-report.txt with full constraint breakdown for judges.

  • Horizontal ASCII tables per dimension
  • Score-based domain ranking (top 3)
  • Tightest passing tier for line budgets
  • File count, language, and line summary

Installation

Setup & Run
in Minutes

Go 1.21+ required. Single binary, zero dependencies. Optional GitHub token for 5,000 req/hour vs 60.

📋 Prerequisites

1

Install Go 1.21+

Download from go.dev/dl and verify with go version

2

Get the Code

Clone or download the project directory and cd into it

3

Optional: GitHub Token

Create a Personal Access Token at GitHub Settings → Developer settings

🚀 Quick Start

# Navigate to project
cd co-check
# Build the binary
go build -o co-check.exe .
# Run with a GitHub URL
.\co-check.exe https://github.com/owner/repo
# Or run interactively
go run .
GitHub URL: https://github.com/...

🔑 Token Setup (All Platforms)

# Windows PowerShell
$env:GITHUB_TOKEN = "ghp_..."
.\co-check.exe https://github.com/...
# Windows CMD
set GITHUB_TOKEN=ghp_...
co-check.exe https://github.com/...
# Linux / macOS
export GITHUB_TOKEN=ghp_...
./co-check https://github.com/...
